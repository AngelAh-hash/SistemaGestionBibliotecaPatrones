# 📚 Sistema de Gestión de Biblioteca

## Descripción

Proyecto desarrollado en Java como trabajo final del curso Diseño de Patrones.

El sistema permite gestionar:

- Registro de usuarios
- Registro de libros
- Préstamos
- Devoluciones
- Reportes
- Notificaciones

## Tecnologías

- Java
- Apache NetBeans
- JDK 17
- JUnit 5
- Git
- GitHub

## Patrones implementados

### Creacionales
- Singleton
- Factory Method
- Builder

### Estructurales
- Facade
- Adapter
- Decorator
- Composite

### Comportamiento
- State
- Observer
- Command
